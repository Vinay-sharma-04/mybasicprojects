# Python Mini Projects Collection

This repository contains a collection of beginner-friendly Python mini projects.
These projects are created to improve Python fundamentals, logic building,
and problem-solving skills.

--------------------------------------------------

Projects Included

1. Simple Calculator with History

A command-line calculator that performs basic arithmetic operations and
stores calculation history in a text file.

Features:
- Supports +, -, *, / operations
- Saves calculation history in a file
- View history using "history" command
- Clear history using "clear" command
- Handles invalid input and division by zero

Concepts Used:
- File handling
- Functions
- Loops
- Conditional statements
- User input handling

--------------------------------------------------

2. Fake News Headline Generator

A fun Python project that generates random and humorous fake news headlines
using predefined subjects, actions, and places.

Features:
- Random headline generation
- Uses infinite loop until user exits
- Interactive user input

Concepts Used:
- Lists
- random module
- Loops
- String formatting

--------------------------------------------------

3. Password Guessing Game

An interactive game where the user guesses a secret password based on the
selected difficulty level.

Features:
- Easy, Medium, and Hard difficulty levels
- Random word selection
- Hint system using underscores (_)
- Countsnumber of attempts

Concepts Used:
- Lists
- Loops
- Conditional statements
- String comparison
- random module

--------------------------------------------------

4. Rock Paper Scissors Game

A classic Rock Paper Scissors game where the user plays against the computer.

Features:
- User vs Computer gameplay
- Random computer choice
- Displays result (Win / Lose / Draw)
- Simple and interactive

Concepts Used:
- Conditional statements
- Loops
- random module
- User input handling

--------------------------------------------------

5. Restaurant Ordering System

A simple command-line based restaurant ordering system using Python.
It allows users to select food items from a predefined menu and
calculates the total bill amount.

Features:
- Displays menu with prices
- Allows ordering of multiple items
- Calculates total order amount
- Handles invalid item selection

Concepts Used:
- Dictionary
- Conditional statements
- User input handling


Technologies Used
- Python 3
- Command Line Interface (CLI)
